== Description ==

Easily reply to BuddyPress notifications from your email, instead of logging into the website every time! Your reply will automatically go back to the website. Reply to BuddyPress notifications by email, from messages, activity posts, bbPress forums, and WordPress comments.

== Installation ==

1. Make sure BuddyPress is activated.
2. Visit 'Plugins > Add New'
3. Click 'Upload Plugin'
4. Upload the file 'buddyboss-reply-by-email.zip'
5. Activate 'BuddyBoss Reply by Email' from your Plugins page.

= Configuration =

1. Visit 'BuddyBoss > Reply by Email' and configure your settings.
2. Sign up for incoming mail service at cloudmailin.com or sendgrid.com.
