Turn your BuddyPress messaging into a state of the art inbox.

== Description ==

BuddyBoss Inbox enables attachments, labels, drafts, auto-save and a visual editor right in your BuddyPress Messages.

== Installation ==

1. Make sure BuddyPress is activated.
2. Visit 'Plugins > Add New'
3. Click 'Upload Plugin'
4. Upload the file 'buddyboss-inbox.zip'
5. Activate BuddyBoss Inbox from your Plugins page.

= Configuration =

1. Enable 'Private Messaging' at 'Settings > BuddyPress > Components'
2. Visit 'BuddyBoss > Inbox' and select your desired options.
